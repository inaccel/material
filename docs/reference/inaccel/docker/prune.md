## Description

Clean all unused data (InAccel containers, images and network)

## Usage

```text
inaccel docker prune
```

## Options

| Name, shorthand | Default | Description                    |
| :-------------- | :-----: | :----------------------------- |
| ` --force, -f ` |         | Do not prompt for confirmation |
