## Description

Set the InAccel Coral license

## Usage

```text
inaccel config license <key>
```
Where `<key>` is the InAccel Coral license key.

## Parent command

| Command                        | Description                                                    |
| :----------------------------- | :------------------------------------------------------------- |
| [` inaccel config `](index.md) | Configures bitstream repositories and coral container settings |
